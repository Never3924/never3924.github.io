ヘルプ

ルール

  • ライフゲームの初期ボードを4x4サイズで好きなように書くことができる
  • セルが誕生すればするほどスコアが増える
  • 目指せハイスコア

遊び方

  1. 左の小さいキャンバスに初期状態を書き込む
  2. 開始ボタンをクリックする
  3. 右のキャンバスでライフゲームが500世代分進行するので、見守る
  4. 結果が左下に出るので、一喜一憂する

その他

  • ハイスコアをクリックするとハイスコアを取ったときの初期ボードで現在のボードが上書きされる
  • このハイスコアとボードの情報はリロードしても保持される
  • クリアボタンで初期ボードを全削除することができる
